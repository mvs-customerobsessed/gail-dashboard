# ACORD 25 - Certificate of Liability Insurance

## Purpose
The most commonly used certificate form. Provides evidence of liability coverage to third parties (certificate holders).

## Supported Coverage Types
- Commercial General Liability (CGL)
- Commercial Automobile Liability
- Umbrella/Excess Liability
- Workers Compensation & Employers Liability

## When to Use
- Certificate holder requests proof of liability coverage
- Vendor/contractor requirements
- Lease agreements requiring liability evidence
- Any request for "COI" without further specification (default form)

## Key Sections

### 1. Producer
Insurance agency/broker information:
- Name and full address
- Contact phone, fax, email
- Producer license number (if required by state)

### 2. Insured
Policyholder information:
- Full legal name (as it appears on policy)
- Mailing address
- DBA names if applicable

### 3. Insurers Affording Coverage
Up to 4 insurers can be listed (A, B, C, D):
- Full insurer legal name
- NAIC code (5-digit identifier)
- AM Best rating (optional but recommended)

### 4. Coverages
Grid showing each coverage type with:
- Type of coverage
- Policy number
- Effective and expiration dates
- Limits of liability

### 5. Description of Operations
Special provisions and project information:
- Additional insureds
- Waiver of subrogation
- Primary/non-contributory status
- Project description
- Locations covered

### 6. Certificate Holder
Entity receiving the certificate:
- Full legal name
- Complete mailing address
- Project reference (if applicable)

### 7. Cancellation
Notice provisions:
- Standard: 30 days written notice
- Non-payment: 10 days notice
- Note: Certificate holder is notified, not guaranteed coverage

## Important Rules

1. **Certificate cannot extend coverage** - Only provides evidence of existing coverage
2. **Cannot modify policy terms** - Policy language governs, not certificate
3. **Additional insured must be in policy** - Note endorsement number in Description
4. **Dates cannot exceed policy period** - Certificate expires when policy expires
5. **Limits shown must match policy** - Cannot inflate or misrepresent limits

## Common Mistakes to Avoid

- Listing incorrect insured name
- Using expired policy dates
- Omitting NAIC codes
- Forgetting to note additional insured status
- Incorrect limit amounts
- Missing producer information
