# ACORD 24 - Certificate of Property Insurance

## Purpose
Provides evidence of property insurance coverage to third parties. Used when property coverage needs to be documented separately from liability.

## Supported Coverage Types
- Building Coverage
- Business Personal Property (BPP)
- Inland Marine
- Equipment Floater
- Installation Floater
- Builders Risk

## When to Use
- Standalone property coverage certificate request
- Inland marine coverage evidence
- Equipment or contractor's tools coverage
- When ACORD 25 is not appropriate (no liability coverage involved)

## Key Sections

### 1. Producer
Insurance agency/broker information

### 2. Insured
Named insured on the property policy

### 3. Company Affording Coverage
Property insurer name and NAIC code

### 4. Location/Description
Property location and description of covered property

### 5. Coverages
- Coverage type (Building, BPP, Inland Marine)
- Policy number
- Effective/Expiration dates
- Coverage amount
- Deductible
- Perils covered (Named Perils, Special/All Risk)

### 6. Mortgagee/Loss Payee
If applicable, the entity with financial interest

### 7. Certificate Holder
Entity receiving the certificate

## Important Rules
- Cannot extend coverage beyond policy terms
- Property must be specifically covered under policy
- Mortgagee/loss payee clauses must be endorsed to policy
- Replacement cost vs. actual cash value should be noted
