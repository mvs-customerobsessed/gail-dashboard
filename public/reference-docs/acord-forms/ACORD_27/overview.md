# ACORD 27 - Evidence of Property Insurance

## Purpose
Provides evidence of property insurance to lenders and mortgagees. Specifically designed for residential or simple commercial property situations.

## Supported Coverage Types
- Dwelling/Building Coverage
- Personal Property
- Loss of Rents/Business Income

## When to Use
- Lender requires proof of property insurance
- Mortgage company requests evidence of coverage
- Residential property insurance evidence
- Simple commercial property for lender

## Key Sections

### 1. Agency Information
Insurance agency/broker details

### 2. Insured
Property owner/policyholder name

### 3. Insurance Company
Carrier providing coverage

### 4. Property Information
- Property address/location
- Type of property
- Policy number
- Coverage amounts

### 5. Coverage Details
- Dwelling/Building amount
- Personal property amount
- Deductible
- Coinsurance percentage
- Valuation (RC or ACV)

### 6. Mortgagee Information
- Mortgagee name
- Loan number
- Mortgagee clause endorsement

## Important Notes
- Used primarily for mortgage/lender requirements
- Simpler than ACORD 28 for basic property coverage
- Must include mortgagee clause if required
- Shows loss payee status on policy
